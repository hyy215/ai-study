LLM 采样参数可视化

探索 Temperature, Top-K, 和 Top-P 如何影响 Token 的概率分布。

当前输入 (Prompt):

"人工智能将改变我们的 [?]"

原始分数与初始概率 (Raw Logits & Initial Prob):

展示模型输出的原始 Logits 分数，以及经 Softmax (T=1.0) 转换后的初始概率。
(注意：Logits 的微小差异经过 Softmax 指数运算后会被显著放大)

控制随机性。低 = 确定性高，高 = 更有创意/随机。

限制仅从前 K 个 Token 中选择。0 表示禁用。

选择累积概率 >= P 的 Token 集合。

工作原理:

  1. Logits: 模型输出的原始分数。
  2. Temperature: 在 Softmax 之前缩放 Logits。 logit / temp。
  3. Softmax: 将缩放后的 Logits 转换为概率（总和为 1）。
  4. Top-K / Top-P: 过滤掉低概率的 Token，然后重新归一化剩余的概率。
